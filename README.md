# WK-portal

Standalone project for WK Portal. This repository is intentionally separate from the website project.

## Run locally

```bash
npm install
npm run dev
```

## Structure

- `src/components` shared UI components
- `src/features` feature modules
- `src/pages` page-level composition
- `src/services` API/data layer
- `src/styles` global styles and tokens
- `tests/unit` unit tests
- `tests/e2e` end-to-end tests
